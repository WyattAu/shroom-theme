Shroom Space Theme

A dark VS Code color theme with accessibility-focused variants. 7 theme configurations covering standard dark, light, and color-vision-deficiency adapted palettes.

code --install-extension wyattau.shroom-space-theme

Available Themes

Shroom Space

dark

Base dark theme. Purple/teal/green palette with warm amber constants.

Shroom Space Light

light

Light variant with warm earth-tone backgrounds and muted accents.

Deuteranopia

dark

Adapted for deuteranopia (green-weak CVD). Blue-shifted additions, avoided red-green confusion.

Protanopia

dark

Adapted for protanopia (red-blind CVD). Coral-shifted errors with blue additions.

Tritanopia

dark

Adapted for tritanopia (blue-yellow CVD). Pink/cyan-shifted accents with green strings.

Monochrome

dark

Grayscale palette. Error indicators use luminance contrast. Maximum text clarity.

High Contrast

hc

Maximum contrast ratios. Pure black background. Full WCAG AAA compliance target.

Installation

Search for "Shroom Space" in the VS Code Extensions panel, or install via CLI:

code --install-extension wyattau.shroom-space-theme

Select a theme via Ctrl+K Ctrl+T (macOS: Cmd+K Cmd+T).

Manual Installation

  1. Download the latest .vsix from releases.
  2. Run: code --install-extension shroom-space-theme-*.vsix
