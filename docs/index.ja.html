Shroom Space Theme

アクセシビリティ重視のバリアントとランタイム機能を備えた VS Code カラーテーマ。7つのテーマ設定、ユーザー設定可能なアクセントカラー、ダーク/ライト自動切替、WCAG 2.1 AA 準拠、9つのエクスポート形式に対応。

code --install-extension wyattau.shroom-space-theme

テーマ一覧

Shroom Space

dark

ベースダークテーマ。紫/ティール/グリーンのパレットに、ウォームなアンバーの定数を組み合わせ。

Shroom Space Light

light

ライトバリアント。ウォームなアーストーンの背景とミュートされたアクセント。

Deuteranopia

dark

2型色覚（緑色弱）向けに適応。ブルーシフトで補完、赤緑の混同を回避。

Protanopia

dark

1型色覚（赤色盲）向けに適応。コーラルシフトのエラー表示にブルーの補色を追加。

Tritanopia

dark

3型色覚（青黄色覚異常）向けに適応。ピンク/シアンシフトのアクセントにグリーンの文字列。

Monochrome

dark

グレースケールパレット。エラー表示は輝度コントラストを使用。最大のテキスト明瞭度。

High Contrast

hc

最大コントラスト比。純黒背景。WCAG AAA 完全準拠を目標。

インストール

VS Code の拡張機能パネルで "Shroom Space" を検索するか、CLI でインストール：

code --install-extension wyattau.shroom-space-theme

Ctrl+K Ctrl+T（macOS：Cmd+K Cmd+T）でテーマを選択。

手動インストール

  1. リリースページから最新の .vsix をダウンロード。
  2. 実行：code --install-extension shroom-space-theme-*.vsix

開発

前提条件：Node.js 22+、npm 10+、VS Code 1.105.0+。

コマンド説明
npm run compileTypeScript コンパイル
npm run lintESLint 厳格ルール
npm run validateテーマ JSON バリデーション
npm testフルパイプライン（コンパイル + lint + バリデーション + 92 テスト）

機能

  • WCAG 2.1 AA コントラスト準拠（全7テーマ）
  • ビジュアルリグレッションテスト（7つの Playwright テスト）
  • マルチエディタエクスポート（9形式：TextMate、JetBrains、Vim、Windows Terminal、iTerm2、Warp、Alacritty、Kitty、CSS）
  • テーマごとに943カラートークン（既知の VS Code API の100%をカバー）
  • テーマごとに81の tokenColors エントリ
  • テーマごとに57の semanticTokenColors ルール
