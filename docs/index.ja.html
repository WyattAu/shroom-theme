Shroom Space Theme

アクセシビリティ重視のバリアントとランタイム機能を備えた VS Code カラーテーマ。7つのテーマ設定、カスタマイズ可能なアクセントカラー、ダーク/ライト自動切り替え、WCAG 2.1 AA 準拠、8つのエクスポート形式に対応。

code --install-extension wyattau.shroom-space-theme

テーマ一覧

Shroom Space

ダーク

ベースダークテーマ。紫/ティール/グリーンのパレットに、温かみのあるアンバー色の定数を組み合わせ。

Shroom Space Light

ライト

ライトバリアント。温かみのあるアーストーンの背景と落ち着いたアクセントカラー。

Deuteranopia

ダーク

緑色弱（Deuteranopia）向けに適応。青色シフトを追加し、赤緑の混同を回避。

Protanopia

ダーク

赤色盲（Protanopia）向けに適応。コーラルシフトのエラー表示に青色を追加。

Tritanopia

ダーク

青黄色盲（Tritanopia）向けに適応。ピンク/シアンシフトのアクセントにグリーンの文字列。

Monochrome

ダーク

グレースケールパレット。エラー表示は輝度コントラストを使用。最大限のテキストの明瞭さ。

High Contrast

ハイコントラスト

最大コントラスト比。純黒背景。WCAG AAA 完全準拠を目標。

インストール

VS Code 拡張機能パネルで "Shroom Space" を検索するか、CLI でインストール：

code --install-extension wyattau.shroom-space-theme

Ctrl+K Ctrl+T（macOS：Cmd+K Cmd+T）でテーマを選択。

手動インストール

  1. releases から最新の .vsix をダウンロード。
  2. 実行：code --install-extension shroom-space-theme-*.vsix

開発

前提条件：Node.js 22+、npm 10+、VS Code 1.105.0+。

コマンド説明
npm run compileTypeScript コンパイル
npm run lintESLint 厳格ルールチェック
npm run validateテーマ JSON バリデーション
npm testフルパイプライン（コンパイル + lint + バリデーション + 92 テスト）

機能

  • WCAG 2.1 AA コントラスト準拠（全7テーマ）
  • ビジュアルリグレッションテスト（7 Playwright テスト）
  • マルチエディタエクスポート（TextMate、JetBrains、Vim、Windows Terminal）
  • テーマあたり 620 カラートークン（VS Code API の 68%）
  • テーマあたり 81 tokenColors エントリ
  • テーマあたり 57 semanticTokenColors ルール
