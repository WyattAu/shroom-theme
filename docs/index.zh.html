Shroom Space Theme

一款具有无障碍功能变体和运行时特性的 VS Code 颜色主题。包含 7 种主题配置，支持用户自定义强调色、自动深/浅色切换、WCAG 2.1 AA 合规，以及 9 种导出格式。

code --install-extension wyattau.shroom-space-theme

可用主题

Shroom Space

dark

基础深色主题。紫/青/绿色调色板，搭配暖琥珀色常量。

Shroom Space Light

light

浅色变体，采用暖大地色背景和柔和的强调色。

Deuteranopia

dark

针对绿色弱（Deuteranopia）色觉缺陷适配。蓝色偏移补充，避免红绿混淆。

Protanopia

dark

针对红色盲（Protanopia）色觉缺陷适配。珊瑚色偏移的错误提示，搭配蓝色补充。

Tritanopia

dark

针对蓝黄色觉缺陷（Tritanopia）适配。粉/青偏移强调色，搭配绿色字符串。

Monochrome

dark

灰度调色板。错误指示器使用亮度对比。最大化文本清晰度。

High Contrast

hc

最大对比度。纯黑背景。完全符合 WCAG AAA 标准。

安装

在 VS Code 扩展面板中搜索 "Shroom Space"，或通过命令行安装：

code --install-extension wyattau.shroom-space-theme

通过 Ctrl+K Ctrl+T（macOS：Cmd+K Cmd+T）选择主题。

手动安装

  1. 从发布页下载最新的 .vsix 文件。
  2. 运行：code --install-extension shroom-space-theme-*.vsix

开发

前置要求：Node.js 24+、npm 10+、VS Code 1.105.0+。

命令说明
npm run compileTypeScript 编译
npm run lintESLint 严格规则检查
npm run validate主题 JSON 验证
npm test完整流水线（编译 + lint + 验证 + 92 项测试）

功能特性

  • WCAG 2.1 AA 对比度合规（全部 7 种主题）
  • 视觉回归测试（7 项 Playwright 测试）
  • 多编辑器导出（9 种格式：TextMate、JetBrains、Vim、Windows Terminal、iTerm2、Warp、Alacritty、Kitty、CSS）
  • 每种主题 943 个颜色令牌（覆盖 100% 的已知 VS Code API）
  • 每种主题 81 个 tokenColors 条目
  • 每种主题 57 条 semanticTokenColors 规则
