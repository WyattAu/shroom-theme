Shroom Space Theme

一款注重无障碍访问的 VS Code 颜色主题，包含多种变体和运行时功能。7 种主题配置，支持用户自定义强调色、自动深浅色切换、WCAG 2.1 AA 合规性，以及 8 种导出格式。

code --install-extension wyattau.shroom-space-theme

可用主题

Shroom Space

深色

基础深色主题。紫色/青色/绿色调色板，搭配暖琥珀色常量。

Shroom Space Light

浅色

浅色变体，采用温暖的大地色调背景和柔和的强调色。

Deuteranopia

深色

针对绿色弱（Deuteranopia）适配。增加蓝色偏移，避免红绿混淆。

Protanopia

深色

针对红色盲（Protanopia）适配。珊瑚色偏移的错误提示，搭配蓝色补充。

Tritanopia

深色

针对蓝黄色盲（Tritanopia）适配。粉色/青色偏移的强调色，搭配绿色字符串。

Monochrome

深色

灰度调色板。错误指示器使用亮度对比。最大化文本清晰度。

High Contrast

高对比度

最大对比度。纯黑背景。完全符合 WCAG AAA 标准。

安装

在 VS Code 扩展面板中搜索 "Shroom Space"，或通过命令行安装：

code --install-extension wyattau.shroom-space-theme

通过 Ctrl+K Ctrl+T（macOS：Cmd+K Cmd+T）选择主题。

手动安装

  1. 从 releases 下载最新的 .vsix 文件。
  2. 运行：code --install-extension shroom-space-theme-*.vsix

开发

前置条件：Node.js 22+、npm 10+、VS Code 1.105.0+。

命令说明
npm run compileTypeScript 编译
npm run lintESLint 严格规则检查
npm run validate主题 JSON 验证
npm test完整流程（编译 + lint + 验证 + 92 项测试）

功能特性

  • WCAG 2.1 AA 对比度合规（全部 7 个主题）
  • 视觉回归测试（7 项 Playwright 测试）
  • 多编辑器导出（TextMate、JetBrains、Vim、Windows Terminal）
  • 每个主题 620 个颜色标记（覆盖 VS Code API 的 68%）
  • 每个主题 81 个 tokenColors 条目
  • 每个主题 57 条 semanticTokenColors 规则
